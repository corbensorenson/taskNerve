# TaskNerve Contributing Ideas

Project: taskNerve

## UX Ideas
- Add in-editor markdown shortcuts palette and quick formatting help.
- Add a compact "controller health" badge per project row.
- Add optional "focus project" mode for single-project deep work.

## Performance Ideas
- Reduce bridge roundtrips for sidebar hydration via batched endpoints.
- Add scoped memoization for project row derived status fields.
- Add host-event subscriptions for rates/resources/thread updates wherever possible.

## Reliability Ideas
- Add startup bridge readiness probe with soft retry before first editor read.
- Add "recover controller" action history with one-click audit output.
- Add safe diagnostics export for crash or blank-screen incidents.

## Workflow Ideas
- Add per-project deterministic import checklist progress UI.
- Add issue-ingest candidate review queue with reject reasons presets.
- Add task queue maintenance policy presets (aggressive/balanced/conservative).
